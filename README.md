The R script (and a markdown document) contained with this repository will help you process point count data uploaded in eBird and create species accumulation curve, and basic abundance and occurrence tables for each species. It also contains an R script (and markdown document) containing benchmarking test for serialised and paralellised spp.est functions from two different R packages. 

Code and documentation authored by Paul Pop.

This research was carried out under the BIRD lab, ATREE, Bengaluru (PI: Rajkamal Goswami).
